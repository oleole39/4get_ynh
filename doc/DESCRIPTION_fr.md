## Fonctionnalités
1. Rotation de proxys configurable pour chaque scraper
2. Filtres de recherche
3. Protection contre les robots (si activé)
4. Interface qui ne nécessite pas Javascript
5. Téléchargement de favicons via cache et proxy d'images
